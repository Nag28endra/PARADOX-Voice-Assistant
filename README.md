# PARADOX

PARADOX is a Windows-based voice assistant that uses a PyTorch neural network for intent classification, speech recognition for microphone input, and PyQt5 for a simple GUI.

## Features

- Voice-based commands for greetings, time, date, search, and entertainment actions
- Windows app launching and closing shortcuts for Notepad, Microsoft Word, Excel, PowerPoint, Brave, and Command Prompt
- News headlines via the configured News API endpoint
- GUI launcher with animated assets

## Project files

- `paradox.py` – main application entry point and PyQt5 GUI wiring
- `Task.py` – task handlers for commands that need input or system actions
- `Listen.py` – microphone capture and speech recognition
- `speak.py` – text-to-speech output
- `Train.py` – trains the intent-classification model
- `NeuralNetwork.py` – text preprocessing helpers
- `Brain.py` – neural network model class
- `paradoxUI.py` – generated PyQt5 UI module
- `intents.json` – command patterns and responses
- `trainData.pth` – trained model weights generated by `Train.py`

## Prerequisites

- Windows 10 or 11
- Python 3.10 or 3.11 (the project has been tested with Python 3.12 in this workspace)
- Microphone and speakers/headphones
- Microsoft Office and/or Brave browser if you plan to use the app-launch commands

## Installation

1. Open a terminal in the project root.
2. Create and activate a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

3. Install the dependencies:

```powershell
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

## Environment setup

The project reads image paths, executable paths, and the News API URL from `.env`.

Create or verify `.env` contains values similar to the following:

```env
main_image="D:/MyProjects/PARADOX/GUI/main.gif"
title_image="D:/MyProjects/PARADOX/GUI/title.jpg"
radiohalo_image="D:/MyProjects/PARADOX/GUI/radiohalo.gif"
motion_sphere_image="D:/MyProjects/PARADOX/GUI/motion-sphere.gif"
radar_image="D:/MyProjects/PARADOX/GUI/radar.gif"
200w_image="D:/MyProjects/PARADOX/GUI/200w.gif"
reload_image="D:/MyProjects/PARADOX/GUI/reload.gif"
alien_image="D:/MyProjects/PARADOX/GUI/alien.gif"
initiating_image="D:/MyProjects/PARADOX/GUI/initiating.gif"
notepad_path="C:\Program Files\Notepad++\notepad++.exe"
ms_word="C:\Program Files\Microsoft Office\root\Office16\WINWORD.EXE"
news_api="http://newsapi.org/v2/top-headlines?sources=techcrunch&apiKey=YOUR_API_KEY"
```

> Update the paths to match your local machine and set a valid `news_api` value if the headlines feature is required.

## Training the model

If you want to regenerate the trained model, run:

```powershell
python Train.py
```

This writes the updated model to `trainData.pth`.

## Running the application

From the project root:

```powershell
python paradox.py
```

The GUI will open and the assistant will start listening after you click the Run button.

## Common issues

- `PyAudio` or microphone access errors: ensure your microphone is connected and Windows microphone permissions are enabled.
- `pyttsx3` errors: confirm you are running on Windows and have a valid speech engine available.
- `Notepad++` or Office paths are wrong: update the paths in `.env`.
- GUI image paths are wrong: confirm the files exist under `GUI/` and update `.env` if needed.
- Training may take a few minutes. You can keep the existing `trainData.pth` if you do not need to retrain.

## Notes

- The repository already contains a `requirements.txt` file.
- The assistant is Windows-specific because it uses `pyttsx3` (`sapi5`) and Windows process-launch helpers.
- Some commands rely on browser and Office executables being installed in the standard locations.
